# Pip-OS
My FO4 Pip-Boy inspired OS im making for hackclub stardance

Features:
A welcome screen.
A Note taking app.
An app where you can view the map of the in-game Commonwealth from Fallout 4.
A fun lockpicking minigame.
A working real time clock wtih the year set to 2077.
A little tip (war never changes).
A fake in-game location.
